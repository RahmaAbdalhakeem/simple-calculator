# Calculator Project 

A simple and attractive calculator using HTML, CSS, and JavaScript.

### Features
- Responsive design for all devices (PC and mobile)
- Attractive buttons with hover 
- CLEAR button to reset the screen and = button to calculate
- Google Fonts for buttons and screen
